# Data Understanding

## Dataset

In this project, I am using the German Credit dataset from the UCI Machine Learning Repository.

## Dataset description

The dataset contains information about clients and their credit situation. It includes different types of variables related to personal and financial background.

## Target variable

The target variable tells us wetheer a client is considered a good or bad credit risk.

## Why I chose this dataset

I chose this dataset beacause it is widely used for credit risk classification and it matches the objective of this project.

## What comes next

Before building any model, I need to explore the data, understand the variables and check if there are any issues such as missing values or data imbalance.
